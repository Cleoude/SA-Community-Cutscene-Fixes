> **SA Community Cutscene Fixes (or SACCF) is a modification for Grand Theft Auto: San Andreas that aims to fix and improve all cutscenes and gameplay in a more advanced way.** 

# Requirements:

- Grand Theft Auto: San Andreas v1.0
- Ultimate ASI Loader
- CLEO
- CLEO+
- NewOpcodes
- Open Limit Adjuster
- Effect Loader
- PedSpec
- ModelVariations
- Mobile Hands
